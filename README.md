### 👤 نظام التسجيل (register.html) - **جديد!**
- صفحة تسجيل/دخول احترافية ثنائية اللوحة
- إنشاء حساب مع: الاسم، username، email، مستوى، دولة، صورة رمزية
- تسجيل دخول بـ username أو email
- تشفير كلمات المرور بـ SHA-256 (Web Crypto API)
- التحقق من قوة كلمة المرور بصري
- التحقق الفوري من توافر username
- حفظ الجلسة في localStorage
- عرض عدد الأعضاء المسجلين
- قبول شروط الاستخدام الإلزامي

### ⚠️ صفحة التنبيه (disclaimer.js)
- تُعرض عند أول زيارة للموقع
- تنبيه صريح بالاستخدام الأخلاقي
- حفظ القبول في localStorage

## 🗄️ قاعدة البيانات

### جدول `users`
| الحقل | النوع | الوصف |
|-------|-------|-------|
| id | text | معرف فريد (UUID) |
| username | text | اسم المستخدم (فريد) |
| email | text | البريد الإلكتروني |
| password_hash | text | SHA-256 hash |
| full_name | text | الاسم الكامل |
| level | text | beginner/intermediate/advanced |
| country | text | رمز الدولة |
| bio | text | نبذة مختصرة |
| avatar_url | text | emoji الصورة الرمزية |
| is_active | bool | حالة الحساب |
| is_verified | bool | تحقق البريد |
| completed_lessons | array | الدروس المكتملة |
| completed_scenarios | array | السيناريوهات المكتملة |
| badges | array | الشارات المكتسبة |
| points | number | نقاط التقدم |
| last_login | datetime | آخر تسجيل دخول |

## 🔌 API Endpoints المستخدمة

```javascript
// إنشاء مستخدم
POST tables/users

// التحقق من username/email
GET tables/users?search=USERNAME&limit=5

// تسجيل الدخول (بحث)
GET tables/users?search=EMAIL&limit=20

// تحديث آخر تسجيل دخول
PATCH tables/users/{id}

// عدد المستخدمين
GET tables/users?limit=1
```
